🚫🎮 FUNCIONAMIENTO DEL ANTICHEAT  

El objetivo de este sistema es evitar que los jugadores usen trampas en Halo Custom Edition, asegurando una experiencia justa para todos. 🔥  

🛑 1. Bloqueo de Programas de Trampas con "TopMost" 🖥️  

- El anticheat usa una técnica llamada TopMost, que mantiene una ventana siempre visible y bloquea el acceso a otros programas mientras el juego está corriendo.  
- Esto evita que los jugadores abran programas como inyectores de DLL, que se usan para modificar la memoria del juego y hacer trampas.  

💡 Básicamente, el juego estará protegido porque no podrás abrir herramientas externas mientras juegas.  

📁 2. Carpeta Temporal para Archivos del Juego ⏳  

- Normalmente, el juego guarda sus archivos en Mis Documentos\My Games\Halo CE, pero algunas extensiones como HAC2 Release y Chimera permiten a los jugadores volver a entrar al servidor sin el anticheat.  
- Para evitarlo, el anticheat crea una carpeta temporal y mueve ahí los archivos necesarios del juego.  
- Al cerrar el juego, la carpeta se elimina automáticamente, asegurando que siempre se usen archivos nuevos y sin modificaciones.  

💡 Esto evita que los jugadores usen archivos alterados para saltarse el anticheat.  

🗄️ 3. Base de Datos de Usuarios y Baneos 🔒  

- Cada jugador es identificado con un ID único basado en el hardware de su PC (UUID del BIOS).  
- No es necesario registrarse, el anticheat obtiene este ID de forma automática.  
- Si un jugador hace trampas, se le banea (se le bloquea el acceso al servidor).  

🚫 ¿Por qué no banear por IP?  
  - Las VPNs permiten cambiar la IP fácilmente, lo que haría que los tramposos pudieran volver a entrar sin problema.  
  - Con este sistema, si un jugador es baneado, su ID de hardware queda registrado en la base de datos, y ya no podrá ingresar, aunque cambie de IP o reinstale el juego.  

💡 Esto hace que los baneos sean efectivos y permanentes para los tramposos.  

🔄 4. Actualización y Feedback de la Comunidad 💡  

- El anticheat se actualizará constantemente para bloquear nuevas trampas que aparezcan en la comunidad.  
- También se aceptan sugerencias y mejoras de los jugadores para hacerlo más seguro y efectivo.  

💬 ¿Tienes ideas o sugerencias? ¡Queremos mejorar este sistema contigo! 🚀  

📌 RESUMEN FINAL:  
✅ Bloquea inyectores y programas externos con una ventana de seguridad.  
✅ Usa una carpeta temporal para evitar el uso de archivos modificados.  
✅ Gestiona baneos con un ID único de hardware, evitando el uso de VPNs.  
✅ Se actualiza regularmente y acepta mejoras de la comunidad.  

📢 ¡Con este sistema, el juego será más justo y protegido para todos! 🔥🎮
